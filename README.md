# Transportation Database System

## Project Description
This project is a relational database system developed to manage transportation operations such as passengers, routes, trips, bookings, payments, and staff. The database is designed using normalization principles to ensure data integrity and reduce redundancy.

## Objectives
- Design a normalized transportation database
- Manage passenger, trip, and booking information efficiently
- Support data retrieval through SQL queries

 ## Tools & Technologies
